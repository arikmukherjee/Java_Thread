# 🎓 University Debate Community Website

A **modern, premium, and fully responsive debate community website** built using **HTML, CSS, and JavaScript**.
This project is designed to represent a university-level debate society where ideas clash, perspectives evolve, and voices are heard.

---

## ✨ Features

* 🎨 **Premium UI/UX Design**

  * Clean, elegant academic aesthetic
  * Dark + Gold / Blue + White theme
  * Glassmorphism and smooth animations

* 📱 **Fully Responsive**

  * Optimized for mobile, tablet, and desktop
  * Seamless user experience across devices

* 🧭 **Interactive Navigation**

  * Sticky navbar with blur effect
  * Mobile hamburger menu

* 🎤 **Events & Debates**

  * Showcase upcoming and past debates
  * Interactive debate topics (Agree/Disagree UI)

* 👥 **Members Section**

  * Core team and community profiles
  * Social links integration

* ✍️ **Blog / Articles**

  * Share opinions, debate insights, and write-ups

* 📊 **Animated Statistics**

  * Counters for debates, members, achievements

* 📬 **Contact Form**

  * User-friendly form with validation

* 🌙 **Modern Effects**

  * Smooth scrolling
  * Hover animations & micro-interactions
  * Optional dark/light mode toggle

---

## 🧱 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Flexbox, Grid, Animations)
* **JavaScript (Vanilla)** – Interactivity
